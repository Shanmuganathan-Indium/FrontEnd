String Methods

length

Extracting strings

slice(start,end)

used to slice the value from start to end

substring(start,end)

same like slice

negative taken from the end

substr(start,length)

takes start and length args

replace()

replace the words

  • Does not change string
  • return new string
  • only change the first value

    • SHAN PAPA

    replaceAll()

    replaceAll the words

    hesha shan papa shan

    toUppercase

    convert all lower case to uppercase

    toLowercase

    convert all upper case to lowercase

    concat()

    used to merge the strings

    Note: All js methods are return new string dont modify the string

    trim()

    used to trim unwanted spaced

    trimstart()

    trim the starting space

    trimend()

    trim the space from the end

    padStart()

    used to give the padding to start

    padEnd()

    used to give the padding to end

    To pad a number convert number to string first

    charAt()

    used to return the character at the specified position (index)

    charcodeAt()

    used to return unicode of the character
