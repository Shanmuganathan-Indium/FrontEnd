JSON

json is format to store and transporting data

json used to sent data from server to webpage

    JSON
  1. Java Script Object Notation
  2. light weight and interchange data
  3. ec to describe
  4. language independent

Example

{ "employees":[ {"firstName":"John", "lastName":"Doe"}, {"firstName":"Anna", "lastName":"Smith"}, {"firstName":"Peter", "lastName":"Jones"} ] }

1) Rules

  1. Key and pair values
  2. separate by comma
  3. {} hold object
  4. [] hold arrays

2) Key and value

"name": "shan"

3) Object

{"name": "shan"}

4) Array

{"data"[{"name": "shan"}]}
